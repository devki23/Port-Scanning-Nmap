# Port Scanning using Nmap

## Objective
The objective of this task is to find all the open ports on a target website.
This helps to understand which services are running on the system.

## Target Website
http://testphp.vulnweb.com/

## Tool Used
- Nmap
- Kali Linux

## Task Description
Port scanning is used to identify open ports and running services on a target.
In this task, Nmap tool is used to scan the target website and collect
information about open ports.

## Steps Performed
1. Opened terminal in Kali Linux.
2. Used Nmap to scan the target website.
3. Saved the scan result in a text file.
4. Took screenshots of the scanning process.

## Command Used
nmap testphp.vulnweb.com

## Output
The scan result shows multiple open ports and running services.
The output file and screenshots are attached in this repository.

## Learning Outcome
- Learned how to use Nmap for port scanning.
- Understood the importance of open ports in cybersecurity.
- Gained basic knowledge of network scanning.

This task was performed for educational purposes only
on a legal and intentionally vulnerable website.
